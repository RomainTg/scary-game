+ Scary Game

Un mini jeu d’horreur très simple, créé comme projet personnel.  
Son but principal est de permettre d’identifier mes défauts tout en s’amusant.

+ Objectifs du projet
- Expérimenter avec HTML / CSS / JavaScript
- Travailler les animations simples et le timing
- Booster mon aisance en JavaScript
- Améliorer progressivement le projet au fil du temps

+ Concept
Le jeu plonge le joueur dans une ambiance inquiétante avec une introduction progressive.  
Il ne s’agit pas d’un jeu complet, mais plutôt d’un terrain d’expérimentation.

+ État du projet
Projet en cours / expérimental.  
Le contenu et les mécaniques sont susceptibles d’évoluer.

+ Technologies utilisées
- HTML
- CSS
- JavaScript (vanilla)

> Projet réalisé à des fins d’apprentissage.
